# AptiConduiteML

This project focuses on multimodal machine learning for neuro-visual impairment assessment in simulated driving.

Due to data privacy and research constraints, the full codebase cannot be publicly released.

## My Contributions
- Designed ML pipeline (feature extraction, selection, classification)
- Implemented multimodal data processing (EEG, ECG, Eye tracking, Simulator)
- Developed interpretable models (SHAP, SLAP)

## Tech Stack
Python, Scikit-learn, MongoDB, Signal Processing

## Status
A sanitized / demo version may be released in the future.
